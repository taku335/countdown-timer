# ワイヤーフレーム資料

## 目的
画面構成と要素の配置を把握するためのワイヤーフレーム説明資料です。

## サイト全体構成
- ルーティングはハッシュルーティングで6画面構成。
  - トップページ（`#/`）
  - 時刻通知ページ（`#/time`）
  - 404（その他のパス）
- どのページも最大幅1000px前後の中央レイアウトで、カード風セクションが縦に並ぶ構成。


## 画面1: 研修用時刻通知ページ

### 画面全体のレイアウト（ワイヤー構成）
```
[Hero: タイトル・説明・戻るリンク]
[表示エリア]
  ├─ タイトル入力
  ├─ 現在時刻 / 終了時刻カード
  └─ 残り時間タイマー
[入力フォーム]
  ├─ 終了時刻入力 + クイック設定ボタン
  └─ 終了時の色合いセレクト
```
- 上段にタイマー表示エリア、下段に入力フォームの2ブロック構成。
- 現在時刻と終了時刻はカード表示、残り時間は大きなタイマー表示。
- クイック設定は「10min/15min/30min/60min」ボタン。
- 終了時の色合いをセレクトで変更できる。

## 画面2: 404ページ

### 画面全体のレイアウト（ワイヤー構成）
```
[Hero: エラー見出し・説明・トップ戻るボタン]
```
- ヒーローカードのみで構成。
- タイトル、説明文、トップへの戻るボタンを配置。

## レスポンシブ挙動（ワイヤー補足）
- 900px以下で検索ページの2カラムが1カラムに変化する。
- 900px以下でタスク入力ページの2カラムも1カラムに変化する。
- 600px以下ではカード内パディングとボタン配置が縦並びになる。
